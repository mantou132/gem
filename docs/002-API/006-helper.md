# Helper

| 名称                        | 描述                                                                      |
| --------------------------- | ------------------------------------------------------------------------- |
| `createTheme`/`updateTheme` | 使用 CSS 自定义属性创建主题，使用 JS 进行引用，使用 TS 时有类型检查的功能 |
| `mediaQuery`                | 使用媒体查询查询到到信息以及一些 CSS 媒体查询常量                         |

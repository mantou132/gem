# Utils

一些经常被用到的函数

| 名称             | 描述                                                      |
| ---------------- | --------------------------------------------------------- |
| `css`            | 模版字符串标签函数，用于 CSS 文本语法高亮                 |
| `raw`            | 模版字符串标签函数，用来 HTML 文本语法高亮                |
| `createCSSSheet` | 使用构造函数创建能附加到元素的样式表                      |
| `styled`         | 模版字符串标签函数的集合，用来提供 CSS 规则高亮和 JS 引用 |

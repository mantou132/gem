# 工具函数

一些经常被用到的函数

| 名称             | 描述                            |
| ---------------- | ------------------------------- |
| `styled`         | 提供 CSS 规则高亮               |
| `styleMap`       | 将对象构建成 `style` 属性       |
| `classMap`       | 将对象构建成 `class` 属性       |
| `partMap`        | 将对象构建成 `part` 属性        |
| `exportPartsMap` | 将对象构建成 `exportparts` 属性 |

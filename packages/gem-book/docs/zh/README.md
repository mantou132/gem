## 轻松上手

```bash
# 安装 gem-book
npm i gem-book

# 创建文档
mkdir docs && echo '# Hello <gem-book>!' > docs/readme.md

# 预览文档
npx gem-book docs

# 构建前端资源
npx gem-book docs --build
```

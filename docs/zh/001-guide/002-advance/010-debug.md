# 调试

GemElement 是标准的自定义元素，能在浏览器 DevTools 中检查和调试。也可以安装 [Gem Devtools](https://chrome.google.com/webstore/detail/gem-devtools/lgfpciakeemopebkmjajengljoakjfle) 扩展，它能让你更直观的检查和修改 Gem 元素的数据。

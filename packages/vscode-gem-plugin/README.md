# Gem for VSCode

Improve the development experience of writing [Gem](https://github.com/mantou132/gem) elements.

## Configure

```json
{
  "gem.elementDefineRules": {
    "Duoyun*Element": "dy-*",
    "*Element": "*",
  },
  // Same as emmet configuration
  "gem.emmet": {},
}
```

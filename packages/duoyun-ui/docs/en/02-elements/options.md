# `<dy-options>`

Options component provides a list of selectable options with customizable labels and styles.

## Example

<gbp-example name="dy-options" src="https://esm.sh/duoyun-ui/elements/options">

```json
{
  "style": "width: 240px;",
  "options": [
    {"label": "Option 1"},
    {"label": "Option 2"},
    {"label": "Option 3", "danger": true}
  ]
}
```

</gbp-example>

# `<dy-rolling-number>`

数字滚动动画组件，当数字变化时，每位数字以垂直滚动方式过渡到新值，滚动过程中数字带有模糊效果，开始时缓慢加速，结束时缓慢减速，中途修改数字时动画保持连贯。
